Description:Re-uses the X-3scale-debug header to add additional logging to the nginx log to view:

  1. headers sent to 3scale
  2. Actual request made to 3scale, and
  3. Response from 3scale
This snippet should be added inside the authorization location that is failing.

{% highlight lua linenos=table %} body_filter_by_lua_block{ if ngx.req.get_headers()["X-3scale-debug"] == 'YOUR_PROVIDER_KEY' then local resp = "" ngx.ctx.buffered = (ngx.ctx.buffered or "") .. string.sub(ngx.arg[1], 1, 1000) if ngx.arg[2] then resp = ngx.ctx.buffered end ngx.log(0, ngx.req.raw_header()) ngx.log(0, resp) end } {% endhighlight %}
